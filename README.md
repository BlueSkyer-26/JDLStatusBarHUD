# JDLStatusBarHUD
状态栏提示器

##JDLStatusBarHUD有哪些功能？

* ``` [JDLStatusBarHUD showSuccess:@"成功"]; ```
* ``` [JDLStatusBarHUD showError:@"失败"]; ```
* ``` [JDLStatusBarHUD showLoading:@"加载中...."]; ```
* ``` [JDLStatusBarHUD showMessage:@"哈哈哈哈哈" image:[UIImage imageNamed:@"jdllogo1024"]]; ```
